# PyCalc - GUI Calculator

A GUI-based 4-function calculator built with PySide6, demonstrating state management and widget composition.

## Programs

- `calculator.py` - Object-oriented calculator widget (single calculator)
- `dual_calculator.py` - Displays two calculator widgets side-by-side
- `end_lecture_1.py` - Early implementation with basic functionality (+, -, =)
- `end_lecture_2.py` - Expanded implementation with all operators (+, -, *, /)

## Running

```bash
uv run pycalc
uv run dual_calc
uv run pycalc-lecture-1
uv run pycalc-lecture-2
```

Or directly:
```bash
uv run python src/pycalc/calculator.py
uv run python src/pycalc/dual_calculator.py
```
