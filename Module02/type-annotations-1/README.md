# Type Annotations Examples

Demonstrates Python type annotation features including type hints, generics, and union types.

## Programs

- `bank_account_no_union.py` - Bank account class with type annotations
- `bank_account_with_union.py` - Bank account using union types
- `generic_stack_class.py` - Generic stack using Python 3.12+ syntax
- `old_generic_stack_class.py` - Generic stack using pre-3.12 syntax
- `interest_no_annotations.py` - Interest calculation without type hints
- `interest_with_type_inference.py` - Interest calculation with type inference
- `interest_with_types.py` - Interest calculation with explicit types
- `vector.py` - Vector class with operator overloading and forward references

## Running

```bash
uv run python bank_account_no_union.py
uv run python generic_stack_class.py
uv run python interest_with_types.py
uv run python vector.py
# etc.
```
