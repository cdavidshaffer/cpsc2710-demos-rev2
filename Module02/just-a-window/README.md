# Just a Window - PySide6 GUI Examples

PySide6 demonstration programs showing Qt widget basics, layouts, and signals.

## Programs

- `main.py` - Basic window with various widgets (labels, buttons, checkboxes, radio buttons, line edits, combo boxes)
- `with_layout.py` - Same widgets organized with proper layouts (VBox, HBox, Grid, Form)
- `with_signals.py` - Adds signal/slot connections for interactive behavior

## Running

```bash
uv run python main.py
uv run python with_layout.py
uv run python with_signals.py
```
